(A) Installation & Settings		                    [Optional]
-----------------------------------------------------------------------------------------------------------------------
1. Install Ananconda or Miniconda			        https://www.anaconda.com                            [~ 722 MB]
                                                    https://docs.conda.io/en/latest/miniconda.html      [~ 46 MB]
2. Create 'pythonlab' folder 	                    $ ~/Projects/pythonlab
3. Save the requirements.txt 	
4. Create a new conda environment	                $ conda create --name pythonlab python=3.9 --file requirements.txt
5. Activate the new environment		                $ conda activate pythonlab

Alternatively,
1. Create virtual environment                       $ python3 -m venv pythonlab                         
2. Install libraries                                $ pip3 install -r requirements.txt
3. Activate the environment                         $ source pythonlab/bin/activate

Note:
Installation & Basics of Python		                Refer Python Primers
Python version 	                                    3.6>=Python<=3.9
